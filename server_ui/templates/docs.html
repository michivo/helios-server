Dokumentation

Das Helios Wahlsystem basiert auf fortgeschrittenen kryptographischen Methoden, die einem Peer-Review unterzogen wurden. Durch die eingesetzten Methoden ist es möglich, dass deine Stimme geheim bleibt, während du selbst mithilfe des smarten Stimmzettel-Trackers stets überprüfen kannst, dass dein Stimmzettel korrekt verarbeitet und gezählt wurde:

  • Technische Dokumentation (auf Englisch)
  • In Peer-Reviews überprüftes Paper (Usenix Security 2008)
  • Pyrios, eine in Go geschriebene Bibliothek von Tom Roeder, mit der Helios-Abstimmungen geprüft werden können.

Helios has been covered in the press:

  • Harvard Magazine, April 2010
  • Ynet, Israeli Press, June 2009
